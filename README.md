# Message Hub
  A message board application by Matthew Farley

### Dependencies
Ruby v2.3.3  
Rails v4.2.8

### Installation
- `git clone https://github.com/MatthewTFarley/message_hub.git`
- `bundle install`

### Customizations
* Haml for view templating
* RSpec for testing

### Testing
Execute specs with `rspec`
